## 🔧 MÉTHODE 2 : Installation via Fichier APK

Si vous préférez un fichier à télécharger et installer manuellement.

### Étapes :

1. **Téléchargez le fichier APK**
   - Fichier : `MaPharmacie_v1.0.0.apk`
   - Taille : ~5 MB

2. **Autorisez les installations de sources inconnues**
   - Allez dans **Paramètres** → **Sécurité**
   - Activez **"Sources inconnues"** ou **"Installer depuis des sources inconnues"**

3. **Installez l'APK**
   - Ouvrez le gestionnaire de fichiers
   - Naviguez jusqu'au fichier `MaPharmacie_v1.0.0.apk`
   - Appuyez dessus pour installer
   - Confirmez l'installation

4. **L'application est prête!**
   - Elle apparaîtra dans votre liste d'applications
   - Lancez-la depuis le menu des applications

## 🆘 Dépannage

### L'application ne s'installe pas (Méthode 1)
- Vérifiez votre connexion Internet
- Essayez avec un autre navigateur
- Videz le cache du navigateur
- Redémarrez votre téléphone

### L'APK ne s'installe pas (Méthode 2)
- Vérifiez que vous avez activé "Sources inconnues"
- Assurez-vous d'avoir assez d'espace disque (au moins 50 MB)
- Téléchargez le fichier complètement avant d'installer
- Essayez de redémarrer votre téléphone

### Le QR Code WhatsApp ne fonctionne pas
- Assurez-vous que WhatsApp est installé
- Vérifiez votre connexion Internet
- Scannez le code avec l'appareil photo de votre téléphone

### L'application est lente
- Videz le cache de l'application
- Redémarrez votre téléphone
- Vérifiez votre connexion Internet
