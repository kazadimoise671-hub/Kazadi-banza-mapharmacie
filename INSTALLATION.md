## 🚀 Installation sur Android (itel version 13)

### Méthode 1 : Installation via navigateur (Recommandée)

1. **Ouvrez Chrome, Firefox ou tout navigateur sur votre téléphone itel**
2. **Accédez à l'application** : Entrez l'URL fournie dans la barre d'adresse
3. **Installez l'application** :
   - Appuyez sur le menu (⋮) en haut à droite
   - Sélectionnez **"Installer l'application"** ou **"Ajouter à l'écran d'accueil"**
   - Confirmez l'installation

4. **L'application apparaîtra sur votre écran d'accueil** comme une véritable application native

### Méthode 2 : Raccourci sur l'écran d'accueil

1. Ouvrez le navigateur et accédez à l'application
2. Appuyez sur le menu (⋮)
3. Sélectionnez **"Ajouter à l'écran d'accueil"**
4. Confirmez - l'application s'ajoute à votre écran d'accueil

## 📖 Guide d'utilisation

### 🏠 Page d'accueil

- Consultez tous les médicaments disponibles
- Filtrez par catégorie (Douleur, Fièvre, Toux, Digestion)
- Cliquez sur un médicament pour voir les détails
- Ajoutez les produits à votre panier

### 🛒 Panier

- Visualisez tous les articles ajoutés
- Modifiez les quantités
- Supprimez des articles
- Consultez le total avec les frais de livraison (2000 FC)

### 💳 Paiement

- Remplissez vos informations de livraison
- Choisissez votre méthode de paiement :
  - **Paiement à la livraison**
  - **Paiement via WhatsApp** (scannez le QR code)
  - **Paiement mobile**
- Confirmez votre commande

### 📞 Contact WhatsApp

Un **QR code WhatsApp** est disponible dans la section paiement :
- **Scannez le code** avec votre téléphone
- **Vous serez redirigé vers WhatsApp** avec le numéro : **+243 993 178 961**
- **Envoyez votre message** pour confirmer votre commande

### 👤 Profil

- Consultez vos informations personnelles
- Visualisez l'historique de vos commandes
- Mettez à jour vos coordonnées

## ⚙️ Dépannage

### L'application ne s'installe pas
- Assurez-vous que votre navigateur est à jour
- Vérifiez que vous avez une connexion Internet stable
- Essayez avec un autre navigateur (Chrome, Firefox)

### Le QR code WhatsApp ne fonctionne pas
- Assurez-vous que WhatsApp est installé sur votre téléphone
- Vérifiez votre connexion Internet
- Scannez le code avec l'appareil photo de votre téléphone

### L'application est lente
- Videz le cache de votre navigateur
- Redémarrez votre téléphone
- Vérifiez votre connexion Internet

### Impossible de passer une commande
- Remplissez tous les champs obligatoires
- Vérifiez votre numéro de téléphone
- Assurez-vous que vous avez des articles dans votre panier
